コマンドプロンプトとは？　コマンドプロンプトを起動するには？

  • Windowsをコマンド（命令文）で操作する為のインターフェースです。
  • コマンド プロンプト: よく寄せられる質問 - Windows ヘルプ

コマンドプロンプトでpythonコマンドが効かない場合は？

  • Pythonのインストールや環境変数PATHの設定が失敗している可能性があります。（参考：1.2_Pythonをインストールする）

GetWebData.pyが実行できない場合は？

  • BeautifulSoupがインストールされていない場合があります。pipでBeautifulSoupをインストールして下さい。

pip自体のインストール方法は？

  • コマンドプロンプトで「python -m ensurepip」と入力・実行します。

  • Python3.4で追加されたensurepipの紹介 - 君は牛を二頭持っている

  • Python3.4からpipが標準インストーラに！？ - Qiita
  • windows環境のPython3.4でpipをつかってパッケージをインストールする - 技術ブログ、あきらめました

pipでBeautifulSoupをインストールするには？

  • コマンドプロンプトで「pip install beautifulsoup4」と入力・実行する。

Pythonコマンドを定期的に自動で実行するには？

  • タスクのスケジュールを設定する - Windows ヘルプ
