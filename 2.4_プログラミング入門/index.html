  • Dive Into Python 3 日本語版

pipとは？

  • Pythonのパッケージ・モジュールの定番管理ツールです。
  • パッケージ等のインストール・アンインストールや更新が簡単に行えるようになります。

pipをインストールするには？

  • コマンドプロンプトで「python -m ensurepip」と入力・実行します。

  • Python3.4で追加されたensurepipの紹介 - 君は牛を二頭持っている

  • Python3.4からpipが標準インストーラに！？ - Qiita
  • windows環境のPython3.4でpipをつかってパッケージをインストールする - 技術ブログ、あきらめました

Pythonを1コマンドずつ結果を確認しながら実行するには？

  1. コマンドプロンプトで「pip install ipython」と入力・実行する。
  2. 「ipython」と入力・実行する。
  3. Pythonのコマンドを1コマンドずつ入力・実行する。
